Application description:

using create-react-app as a boiler-plate.

Packages used - redux, react-redux, redux-form, axios.

redux - Create redux store.
react-redux - Connect react to redux.
redux-form - Create the form in redux store.
axios - Handle HTTP requests.

using bootstrap.

Build instructions:

- Using npm: $ npm install
             $ npm start
